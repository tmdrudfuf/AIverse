# Quickstart: Project Task Management Foundation

1. Open the Daily Proof office computer project portal.
2. Open Daily Proof, then open the workspace.
3. Select Tasks and press Enter or Space.
4. Verify the three placeholder tasks render with status and priority.
5. Open a task detail and press Enter or Space to record Assign Employee as a placeholder action.
6. Verify Esc navigation returns task-detail to task-list, then workspace, project detail, project list, and closed portal.
