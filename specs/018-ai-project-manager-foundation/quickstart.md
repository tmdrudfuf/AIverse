# Quickstart

## Validation Commands
- npx tsc --noEmit
- npm run build
- npm run lint if available
- git diff --check

## Manual Regression Checklist
1. Open Project Portal.
2. Move through project selection.
3. Open Daily Proof workspace.
4. Open Tasks and Task Detail.
5. Assign an employee manually.
6. Start Work and confirm work-session/activity behavior remains unchanged.
7. Move task to Review and Done.
8. Confirm no project-manager UI is visible and no automatic gameplay action occurs.
