# Quickstart

1. Open Daily Proof workspace.
2. Open Tasks.
3. Assign an employee to a Todo task.
4. Confirm Next Action is Start Work (placeholder).
5. Press Enter/Space.
6. Confirm task status becomes In Progress.
7. Confirm Activity shows "GPT Engineer started placeholder work session".
8. Confirm Work Session shows Provider placeholder, Status running, and Started timestamp.
9. Confirm Move to Review and Mark Done still work.
