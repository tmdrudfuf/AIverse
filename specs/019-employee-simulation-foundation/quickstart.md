# Quickstart

## Validation Commands
- npx tsc --noEmit
- npm run build
- npm run lint if available
- git diff --check
- git diff --cached --check

## Manual Regression Checklist
1. Open Project Portal.
2. Open Daily Proof workspace and Tasks.
3. Open Task Detail and assign an employee.
4. Start Work and confirm work-session/activity behavior remains unchanged.
5. Move task to Review and Done.
6. Confirm no visible NPC sprites, animation, pathfinding, schedule, conversation, or UI changes appear.
7. Confirm existing hidden AI analysis/recommendation/project-manager flows still compile and build.
