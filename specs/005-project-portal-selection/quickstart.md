# Quickstart: Project Portal Selection

## Automated Validation

Run from repository root:

```powershell
npx tsc --noEmit
npm run build
git diff --check
```

## Manual Validation

1. Enter Daily Proof office.
2. Open portal from computer with Space.
3. Confirm Daily Proof is selected by default.
4. Press Down/Up and confirm selection changes.
5. Press Enter on Daily Proof and confirm detail view opens.
6. Press Esc and confirm return to list.
7. Select Portfolio or AI Lab and confirm coming-soon detail appears.
8. In detail view, press Enter/Space and confirm placeholder action only.
9. Press Esc from list and confirm portal closes.
10. Confirm movement and Q/E zoom remain blocked while portal is open.
11. Confirm movement, zoom, computer interaction, and office exit work after portal closes.
12. Confirm no real GitHub/Firebase/API calls, NPCs, simulation, React overlay, or complex dashboard UI were added.
