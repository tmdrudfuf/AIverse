# Quickstart: Employee Status Update

1. Open Daily Proof workspace, then Tasks.
2. Open a task detail and confirm Assigned is None.
3. Assign GPT Engineer.
4. Confirm Task Detail shows Assigned: GPT Engineer (Working).
5. Reopen employee selection and confirm GPT Engineer shows Working.
6. Reassign to GPT QA.
7. Confirm GPT QA shows Working and GPT Engineer returns to Idle if no other loaded task uses GPT Engineer.
