# Requirements Checklist

- [x] Task Detail action is status-aware.
- [x] Todo without assignee opens employee selection.
- [x] Todo with assignee starts placeholder work.
- [x] In Progress moves to Review.
- [x] Review moves to Done.
- [x] Done is Completed and no-op.
- [x] Status transitions record activity.
- [x] Employee status is released on Done when no other loaded assignment exists.
- [x] No AI calls, automation, networking, timers, persistence, or React.
