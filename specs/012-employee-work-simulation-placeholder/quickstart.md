# Quickstart

1. Open Daily Proof workspace.
2. Open Tasks.
3. Open an unassigned task and confirm Next Action is Assign Employee.
4. Assign GPT Engineer.
5. Confirm Task Detail shows GPT Engineer as Working and Next Action is Start Work (placeholder).
6. Press Enter/Space.
7. Confirm the newest Activity entry says GPT Engineer started placeholder work.
8. If the task was Todo, return to Task List and confirm it shows In Progress.
