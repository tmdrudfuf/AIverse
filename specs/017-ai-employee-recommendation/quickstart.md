# Quickstart

## Validation Commands
- npx tsc --noEmit
- npm run build
- npm run lint if available
- git diff --check

## Manual Regression Checklist
1. Open Daily Proof workspace.
2. Open Tasks and move through task selection.
3. Open Task Detail.
4. Open Assign Employee and verify manual employee selection works unchanged.
5. Assign an employee manually.
6. Start Work and confirm work session/activity behavior remains unchanged.
7. Move task to Review and Done.
8. Confirm no recommendation UI is visible and no employee is auto-assigned.
